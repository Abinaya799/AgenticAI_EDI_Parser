# AgenticAI_EDI_Parser

Objective & Scope
•	The Billing Reconciliation Agentic AI System automates the extraction, normalization, and validation of carrier freight invoices. 
•	The MVP focuses on converting EDI-210 invoices into a standardized “Golden Invoice” JSON format, creating a unified data foundation for future audit and posting automation.
•	In-scope (MVP): Parsing EDI-210 and producing validated Golden Invoice JSON.
•	Future phases: Audit, exception management, and ERP posting workflows.
